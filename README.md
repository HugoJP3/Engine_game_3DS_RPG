Este proyecto está actualmente en desarrollo.

Pequeño juego estilo RPG para la Nintendo old 3DS (debería funcionar igualmente en una new 3DS).
Hecho en C++ sin engine externo, trabajando con las librería gráficas y de desarrollo de Homebrew: Citro2D, Citro3D y devkitpro.

Desarrollado sin fines comerciales como hobby :)

Para optener el "game.3dsx" necesitas ejecutar el Makefile con las librerías instaladas, y los paths correctos.

Todas las imágenes y audio las extraje de fuentes externas por lo que no son de mi dominio y están diseñadas para fines no comerciales.
